<h1>🔎 CyberPort Scanner</h1>

<p>
CyberPort Scanner is a web-based cybersecurity tool designed to scan and identify open ports on a target host. 
The project demonstrates the fundamental concept of port scanning used in network security, penetration testing, 
and vulnerability assessment.
</p>

<p>
This tool allows users to input a target IP address or hostname and scan a range of ports to determine which ports 
are open and potentially running services. By analyzing open ports, users can better understand network exposure 
and how systems communicate over a network.
</p>

<p>
The project is built using modern web technologies such as <b>React</b>, <b>Vite</b>, and <b>TypeScript</b>, providing 
a fast, responsive, and interactive interface. CyberPort Scanner is intended for educational purposes and helps 
beginners learn about network reconnaissance and cybersecurity tools.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
<li>Scan target IP addresses or hostnames</li>
<li>Detect open ports within a specified port range</li>
<li>Simple and interactive user interface</li>
<li>Built with modern web technologies</li>
<li>Demonstrates basic network reconnaissance concepts</li>
<li>Fast development environment using Vite</li>

<h2>⚙️ Installation</h2>

<p><b>Clone the repository:</b></p>

<pre>
git clone https://github.com/yourusername/cyberport-scanner.git
</pre>

<p><b>Navigate to the project folder:</b></p>

<pre>
cd cyberport-scanner
</pre>

<p><b>Install dependencies:</b></p>

<pre>
npm install
</pre>

<p><b>Run the development server:</b></p>

<pre>
npm run dev
</pre>

<p><b>Open in browser:</b></p>

<pre>
http://localhost:5173
</pre>
